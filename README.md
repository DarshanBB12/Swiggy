# Swiggy Clone (HTML & CSS)

This project is a **Swiggy Website Clone** created using **HTML and CSS**.
It is a front-end replica of the Swiggy homepage, designed to practice layout design, styling, and responsive web development.

## Project Description

The goal of this project is to recreate the look and feel of the Swiggy website.
It focuses on building a clean UI and understanding how modern food delivery websites are structured.

## Features

* Swiggy-like homepage design
* Navigation bar with logo and menu
* Food category sections
* Restaurant cards layout

## Technologies Used

* HTML5
* CSS3
* Flexbox / Grid

## Project Purpose

This project was created for learning and practice purposes to improve:

* HTML structure
* CSS styling
* Layout design skills
* Front-end development basics
